# Company Logos

This directory contains the company logos displayed on the home page under "Join over 1,000 real estate professionals using NextDeal".

## Required Logo Files

Please save the following logo images in this directory:

1. **brookfield-logo.png** - Brookfield logo (dark blue text on black background)
2. **firstservice-logo.png** - FirstService logo (geometric icon with text, dark green background)
3. **maple-leaf-logo.png** - Red maple leaf logo (stylized red maple leaf on black background)
4. **abstract-logo.png** - Abstract blue diamond/ribbon logo (symmetrical diamond-like shape with blue gradient)
5. **mattamy-homes-logo.png** - Mattamy Homes Wildflowers logo (dark blue oval with "mattamyHOMES" and "WILDFLOWERS" text)

## Logo Specifications

- **Format**: PNG with transparent background (preferred) or matching the design
- **Size**: Logos will be displayed at height of 32px (h-8), width will be auto-adjusted to maintain aspect ratio
- **Styling**: Logos will have grayscale filter and reduced opacity by default, which will be removed on hover

## Image Requirements

- All logos should be high quality and scalable
- Maintain aspect ratios
- Ensure logos are readable at small sizes
- Transparent backgrounds are preferred for better integration
